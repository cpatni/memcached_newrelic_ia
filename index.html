memcached_newrelic_ia by rubyorchard

New Relic Memcached Infrastructure Agent. Use this gem to collect statistics from a given host by running newrelic_ia -a as a daemon. This will contribute System/* metrics to your RPM console which can be viewed with custom dashboards.

memcached_newrelic_ia collects memcached, disk, cpu and IO statistics. Collected stats are sent to the New Relic RPM application. Memcached statistics sampler collects subset of stats provided by memcached daemon. In addition it computes other stats dervied from memcached stats. Also support stats derivatives over time to compute rate etc.

Dependencies

  • New Relic Agent version 2.9.2 or later
  • Gold Subscription for creating and viewing custom dashboards

    • Install

    sudo gem install newrelic_ia
    Once installed, run from the home directory of your Rails application, or create a separate working directory and run 
    newrelic_ia --install
    to create a template newrelic.yml file you can use. Edit this file and substitute your license key and app_name value. specify memcached nodes in memcached-nodes.txt.
